# Edge Analysis — Weather Markets (2026-03-05)

## Методология

1. Получили прогнозы daily max из Open-Meteo API (GFS/ECMWF)
2. Модель: `Normal(forecast_max, σ=3°F / 1.5°C)` (2-day horizon)
3. Сравнили P(бакет) модели с PM ценами

## Паттерн

PM систематически:
1. **Переоценивает** бакет ближайший к "типичной" температуре
2. **Недооценивает** хвосты (как upper так и lower)
3. **Сильно переоценивает** верхний крайний бакет (≥X°F)

Это похоже на anchoring bias: трейдеры ставят на "среднюю" температуру,
не учитывая дисперсию прогноза.

## Ликвидность

- Бакеты с edge 10%+ имеют ликвидность \$300-2,000
- Реалистичная покупка: \$50-200 на бакет без проскальзывания
- При 3-4 бакетах с edge × 10-15 городов = \$150-600 в день exposure
- Expected profit при edge 15%: \$25-90/день
