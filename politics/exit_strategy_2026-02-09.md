# Exit Strategy — Political Portfolio (2026-02-09)

## Общие правила

1. **Проверка smart money** — перед любым решением о выходе запроси актуальный анализ:
   ```bash
   python crypto/smart_money.py <event-slug> --market "<filter>"
   ```
2. **Sell если:** smart money flow развернулся (из NO в YES, из YES в NO), ИЛИ edge исчез
3. **Hold если:** smart money подтверждает, time decay работает на тебя
4. **Никогда** не продавай в панике на спайке — проверь, был ли реальный повод
