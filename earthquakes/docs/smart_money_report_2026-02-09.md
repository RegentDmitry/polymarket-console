# Smart Money Analysis — Earthquake Positions (2026-02-09)

## Методология

Conviction-weighted Smart Money Flow: анализ топ-30 холдеров YES и NO по каждому рынку.
Для каждого холдера получаем историческую прибыльность (realized + unrealized P&L) из всех позиций на Polymarket.

**Вес трейдера** = `profit × conviction × shrinkage`
- **conviction** = доля этого рынка в портфеле трейдера
- **shrinkage** = `trades / (trades + 30)` — фильтр от lucky trades
- **science bonus** = +50% для трейдеров с >30% earthquake/science рынков

**Smart Implied** = `yes_weight / (yes_weight + no_weight)` — вероятность YES по мнению прибыльных трейдеров
**Edge** = Smart Implied - PM Price
